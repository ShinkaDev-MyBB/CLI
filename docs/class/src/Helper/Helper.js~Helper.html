    Helper

    Outputs examples for usage (--help) information.

    Example:

    new Helper().outputHelp(
     [{ command: "-v", description: "Outputs verbose error messages" }],
     "link"
);
// => Examples:
//        $ shinka link -v   Outputs verbose error messages
